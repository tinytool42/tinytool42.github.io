About TinyTool

TinyTool is a small but powerful batch script designed to optimize and manage your Windows system with ease.

It includes temp file cleanup, system info display, startup app management, and more.

Download Versions

Features

  • Clean temp and %temp% folders
  • Display IPv4/IPv6, MAC address, OS info
  • Manage startup programs
  • Run a speedtest via speedtest.net
